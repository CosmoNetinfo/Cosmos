# Come Inserire Cosmos nel tuo Sito 

Per inserire l'agente Cosmos in una pagina del tuo sito web (WordPress o HTML), usa il codice iframe qui sotto.

## Codice Embed

Copia e incolla questo codice nell'HTML della tua pagina:

```html
<iframe 
    src="https://cosmos-three-omega.vercel.app/" 
    width="100%" 
    height="700px" 
    style="border:none; border-radius: 12px; box-shadow: 0 4px 20px rgba(0,0,0,0.1);" 
    title="Cosmos AI Agent">
</iframe>
```

### Note
- Puoi regolare `height` (altezza) a piacimento (es. `800px`).
- Assicurati che il link `src` corrisponda esattamente all'URL del tuo deployment su Vercel (controlla che sia `https://cosmos-pi.vercel.app/` o simile).
